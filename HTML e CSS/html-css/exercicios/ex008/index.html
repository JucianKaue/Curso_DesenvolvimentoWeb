Código-fonte / Pré-Formatação

É possível marcar comandos e codigos fonte como window.document.getElementById() formatando com uma fonte de espaçamento fixo por padrão, que pode ser feito usando a tag <CODE>

Para escrever blocos maiores de código é possivel usar a tag <PRE> junto com a tag <CODE>

        
    numeros = [[], [], []]
    for x in range(0, 3):
        for y in range(0, 3):
            numeros[x].append(int(input(f'Escreva o valor para a posição [{x}, {y}]: ')))
    print('\033[36m=\033[0m'*40)
    for x in range(0, 3):
        for y in range(0, 3):
            print(f'[ {numeros[x][y]:^4} ]', end='')
        print()

Citações simples

É possível marcar citações de outros autores usando a tag <Q>, como por exemplo:

Como diria um sábio: O cumputador é um burro muito rápido

Citações completas

Para citações maiores é melhor usar a tag <BLOCKQUOTE>, como no exemplo a seguir

No livro PHP: a Bíblia, Tim Converse explica o que é PHP.

O PHP é uma linguagem para a criação de scripts para a Web do lado servidor embutidos em HTML, cujo código-font é aberto, e que é compatível com os mais importantes servidores Web (especialmente Apache). O PHP permite incorporar fragmentos de código em páginas HTML normais - código esse que é interpretado à medida que suas páginas são oferecidas aos usuários. O PHP também serve como uma linguagem que "cola", facilitando a conexão de suas páginas Web com o banco de dados do lado servidor.

É possível salvar a fonte do conteúdo na tag blockquote usando o parâmetro cite
